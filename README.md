## 🚀 Featured Projects

### 🏟 [The Green Field](https://github.com/Tahmeed3007/the-green-field) — Sports Turf Booking Platform
> Full-stack booking & operations management platform for a two-turf sports venue (football & cricket) in Chittagong, Bangladesh.

- Customer-facing booking site with real-time slot availability, bKash/cash advance payment flow
- Password-protected admin panel: booking verification, payment collection, expense tracking, PDF/CSV reports
- Role-based staff system (owner / manager / staff) with granular permissions and a full audit log
- Intelligent mixed peak/off-peak pricing — bookings spanning the 5 PM boundary are automatically split and billed correctly

**Stack:** Next.js 14 · TypeScript · Supabase (PostgreSQL) · Tailwind CSS · Recharts · jsPDF · Vercel

---

### 🏢 [FlattyBee](https://github.com/Tahmeed3007/flattybee) — Property Management Platform
> Full-stack property management system for a UK-based agency managing residential and commercial properties in Bangladesh.

- Six role portals: Super Admin, Operations Manager, Field Agent, Landlord, Tenant, Contractor — each with its own dashboard and permissions
- Modules: maintenance tickets with before/after photo enforcement, rent tracking, deposit management, document verification, role-based messaging
- Full audit log, GPS field agent check-ins, dual timezone display (BST + BDT), emergency alert banners

**Stack:** Next.js 16 · TypeScript · PostgreSQL · Prisma · NextAuth v5 (Argon2id) · Tailwind CSS v4 · Recharts · Vercel

---

### 💰 [Fynlo](https://github.com/Tahmeed3007/fynlo) — Personal Finance Tracker
> Personal finance tracker built around a biweekly 60/30/10 budget rule.

- Payday logger with animated card-flip 60/30/10 budget preview
- Expense tracker with category pie chart and monthly trend line
- Loan payoff rings with projected payoff dates, "owed to me" tracker with overdue highlighting
- Automated email reminders via Supabase Edge Functions + Resend cron

**Stack:** Next.js 14 · TypeScript · Supabase (Postgres + Auth + Edge Functions) · Framer Motion · Recharts · Resend · Vercel

### 🍽 [Restaurant Management System](https://github.com/Tahmeed3007/restaurant-management-system) — Full-Stack RMS
> Replaces paper-based order management with a complete digital system for a restaurant.

- Two roles: Admin (menu, inventory, analytics, staff) and Waiter (orders, billing, kitchen screen, reservations)

- Analytics dashboard: revenue vs. costs vs. profit (daily / weekly / monthly), exportable as PDF by email

**Stack:** Python · Flask · MySQL · HTML5 · CSS3 · JavaScript · jQuery · Chart.js · pdfkit
